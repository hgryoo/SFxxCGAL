# SFxxCGAL
A wrapper around CGAL that intents to implement 2D and 3D operations on OGC standards models for Extra Large Data Sets

## Installation

### Prerequisies

- CMake
- Boost
- MPFR
- CGAL
- SFGAL
- STxxL
- libspatialindex

### Supported platform

- GNU/Linux

## Building
```
// for installing Boost, CGAL, SFCGAL, STxxL, libspatialindex automatically
./cppbuild.sh

//
cmake .
```
